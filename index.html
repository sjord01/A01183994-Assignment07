Assignment 07

JavaScript Event Handling, DOM manipulation, and Form Processing

This assignment involves creating an interactive HTML form that includes various input types, such as text fields, a password input, a date picker, and a dropdown menu for course selection. The form will feature custom CSS for styling and JavaScript for validation to enhance the user experience. Key features include a tooltip for the student ID field and a toggle to show or hide the password. Additionally, the form will have restrictions on date inputs, specific validation rules, and error handling to ensure a comprehensive and user-friendly submission process.
