Assignment 07

Custom T-Shirt Designer

This HTML program creates an interactive product page for a t-shirt, allowing users to select color, size, and quantity. It uses jQuery for dynamic content updates, including a product image gallery that changes based on color selection, and real-time price calculation. The program implements form validation, ensuring users select a size before enabling the "Add to Cart" button. Upon form submission, it redirects to a thank-you page with a dynamically generated message summarizing the purchase details.
